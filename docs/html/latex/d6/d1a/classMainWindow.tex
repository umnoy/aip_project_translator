\doxysubsection{Подробное описание}
Главное окно приложения переводчика 

Класс \doxylink{classMainWindow}{Main\+Window} представляет собой основное окно приложения, содержащее все элементы интерфейса для перевода текста. Поддерживает как локальный перевод через ONNX Runtime, так и онлайн-\/перевод через различные сервисы.
